    # 2.列表

    无序列表 语法： 无序列表用 - + * 任何一种都可以

    - 列表内容
+ 列表内容
* 列表内容

注意：- + * 跟内容之间都要有一个空格

3. 列表内容

注意：序号跟内容之间要有空格

-两边加：表示文字居中

    # 9.代码

    单行代码：代码之间分别用一个反引号包起来 代码内容

    代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行

    (```)
    function fun(){
         echo "这是一句非常牛逼的代码";
    }
    fun();
(```)
