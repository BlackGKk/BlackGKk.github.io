    该内容不包含Vuex

    # 1.组件之间的关系

    在进行组件之间的数据共享前，需要先了解组件之间的关系：

    1. 父子关系
    2. 兄弟关系
    3. 后代关系

    每种关系都有对应的通信方式。

    # 2.父子组件之间的数据共享

    父子组件之间的数据共享可分为：

    1. 父 -> 子
    2. 子 -> 父
    3. 父 <-> 子 (双向)

    # 父组件向子组件共享数据

    父组件通过v-bind 属性绑定向子组件共享数据。 同时，子组件用props接受数据。

    // 父组件
<son :msg="message"></son>

data() {
return {
  message: '111'
}
}

    # 3.兄弟组件之间的数据共享

    兄弟组件之间实现数据共享的方案是EventBus Vue3需要借助第三方的包mitt来创建eventBus对象， 从而实现兄弟组件之间的数据共享。如下：

    首先安装mitt依赖包

    npm install mitt@2.1.0
    1

    新建eventBus.js文件用于创建公共的EventBus模块

    import mitt from 'mitt'
const bus = mitt()
export default bus
    1
    2
    3

    在数据接收方，调用bus.on('事件名称'，事件处理函数)方法注册一个自定义事件

    import bus from './eventBus.js'
export default{
data(){ return { count:0 } }
created(){ 
  bus.on('countChange',(count)=> {
    this.count = count
  })
 }
}

    # 4.后代关系组件之间的数据共享

    后代关系指的是父节点组件与其子孙组件。 可以使用provide和inject实现后代关系组件之间的数据共享

    父节点通过provide方法，对其子孙组件共享数据

    data() {
 return {
   color: 'red'
  }
